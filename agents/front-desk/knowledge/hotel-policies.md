# 🏨 酒店运营政策知识库

## 入住政策

### 入住时间
- 标准入住时间: **14:00** 以后
- 提前入住: 视房间可用情况而定，可能收取半天房费
- 延迟入住: 预订保留至当日 **18:00**，超时需联系确认

### 入住要求
- 所有入住客人必须出示 **有效身份证件**（身份证/护照/回乡证）
- 必须完成 **人脸识别验证**
- 同住人需同时登记
- 未成年人须由成年人陪同登记

### 押金政策
- 标准间/豪华房: ¥500
- 套房: ¥1,000
- 总统套房: ¥2,000
- 支持微信、支付宝、银行卡预授权

## 退房政策

### 退房时间
- 标准退房时间: **12:00** 之前
- 延迟退房至 14:00: 免费（视房态）
- 延迟退房至 18:00: 加收半天房费
- 延迟退房至 18:00 以后: 加收一天房费

### 退房流程
1. 客人交还房卡
2. 核查房间内消费（迷你吧、损坏等）
3. 结清费用 → 退还押金
4. 开具发票（如需）

## 紧急处理

### 紧急联系
- 酒店总机: 分机 0
- 值班经理: 分机 8001
- 安保部: 分机 8888
- 急救电话: 120
- 报警电话: 110
- 消防电话: 119

### 紧急情况处理
1. **客人突发疾病**: 立即拨打 120，通知值班经理
2. **火灾警报**: 按紧急疏散流程，引导客人撤离
3. **客人纠纷**: 通知安保部，值班经理到场协调
4. **物品丢失**: 记录信息，通知安保部调查
