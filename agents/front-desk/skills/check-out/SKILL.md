# 🏨 退房办理 (Check-Out)

## 触发条件

当客人提到：退房、办理退房、check out、我要退房、结账。

## 完整流程

### Step 1: 确认房间号

```
🏨 好的，为您办理退房。
请问您的房间号是多少？
```

### Step 2: 查询入住信息

根据房间号查找对应的入住记录，回复：
```
✅ 查询到入住信息：
👤 [客人姓名]
🚪 房间 [房号]
📅 入住 [入住日期] ~ [离店日期]

请将房卡放在前台，我来为您核查账单。
```

没有查到回复：
```
❌ 未查询到 [房号] 房间的入住记录。请确认房间号是否正确。
```

### Step 3: 核查消费账单

列出所有费用：
```
📋 退房账单

🛏️ 房费: ¥[金额]

📦 额外消费:（如有）
  • [类别]: [描述] — ¥[金额]

━━━━━━━━━━━━━━━━
💰 合计: ¥[总金额]

请确认账单是否正确？(确认/有异议)
```

### Step 4: 处理结算

- 如需补缴：询问支付方式（微信/支付宝/现金/银行卡）
- 如需退款：告知原路退回
- 客人有异议：通知值班经理协助

### Step 5: 送别

```
🎉 退房完成！

👤 [客人姓名]
🚪 房间 [房号] 已释放
🔑 房卡已注销
💰 总计 ¥[金额]

感谢您的入住！期待下次再见！🌟
祝您旅途愉快！🚗✨
```

## 模拟数据

模拟模式下，当前入住的房间：
- 1203 房间（标准间，状态：在住）
- 1502 房间（套房，状态：在住）

额外消费类别对照：
- room = 房费
- minibar = 迷你吧
- restaurant = 餐饮
- laundry = 洗衣
- spa = 水疗
- damage = 损坏赔偿

模拟模式下额外消费为空（即仅收取房费）。

## 注意事项

- 退房后房间状态改为"清洁中"
- 对账单有异议时，不要自行处理，告知客人已通知经理
- 遇到系统异常告知"不会影响您的退房"安抚客人
