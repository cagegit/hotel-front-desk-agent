# 🏨 客房服务 (Room Service)

## 触发条件

当客人提到：客房服务、room service、打扫、清洁、维修、送餐、需要毛巾、空调、热水、噪音、投诉。

## 完整流程

### Step 1: 确认房间号

如果已知当前客人房间号，直接问候：
```
🏨 [房号] 房间的客人您好！请问需要什么服务？
```

否则询问：
```
🏨 您好！请问您的房间号是多少？
```

### Step 2: 了解服务需求

```
请问您需要以下哪种服务？

1️⃣ 🧹 客房清洁
2️⃣ 🔧 设施维修（空调/热水/电视等）
3️⃣ 🍽️ 送餐服务
4️⃣ 🧴 补充用品（毛巾/洗漱用品/拖鞋等）
5️⃣ 💬 其他需求

请输入序号或直接描述您的需求：
```

### Step 3: 服务分类

根据客人描述自动分类：

| 关键词 | 类型 | 优先级 |
|--------|------|--------|
| 清洁、打扫、卫生 | 🧹 客房清洁 | 普通 |
| 维修、坏了、空调、热水、漏水 | 🔧 设施维修 | ⚡ 高 |
| 送餐、吃、餐 | 🍽️ 送餐服务 | 普通 |
| 毛巾、洗漱、拖鞋、用品 | 🧴 补充用品 | 低 |
| 投诉、噪音、吵、脏、不满 | 💬 投诉反馈 | 🔴 紧急 |

### Step 4: 创建工单

生成工单号（格式 `SVC-XXXX`），回复：
```
✅ 已为您创建服务工单

📋 工单号: SVC-XXXX
🏷️ 类型: [服务类型]
⚡ 优先级: [优先级]
📝 描述: [客人描述]

我已通知相关部门，[预计时间]内会有工作人员处理。
如有其他需要，请随时呼叫！🙏
```

### Step 5: 通知经理 Agent

使用 `sessions_send` 工具通知经理 Agent（小管）此工单，以便调度人员。

## 预计响应时间

| 条件 | 预计时间 |
|------|---------|
| 紧急 | 10 分钟 |
| 高优先级 | 15 分钟 |
| 送餐 | 30 分钟 |
| 清洁 | 20 分钟 |
| 其他 | 15 分钟 |

## 注意事项

- 描述太简短时（如只输入序号），追问详细需求
- 投诉类自动标记为紧急
- 工单创建后应同时通知经理 Agent
