# 📷 视觉管道 (Vision Pipeline)

## 触发条件

当系统检测到摄像头画面中有人体出现时自动激活，或用户提到：查看监控、谁在前台、检测来宾。

## 功能说明

本技能使用 macOS 摄像头进行：
1. **人体检测** — 检测前台区域是否有客人
2. **人脸识别** — 与已注册客人比对
3. **主动问候** — 识别到新来宾时主动打招呼
4. **防重复** — 避免重复问候同一位客人

## 笔记本测试说明

> ⚠️ 在没有外接摄像头或不需要视觉功能的本地测试环境中，此技能的视觉检测功能不会主动触发。客人可以直接通过文字对话触发入住、退房等其他技能。

## 工作流程（有摄像头时）

1. 持续监测摄像头画面
2. 检测到人体 → 尝试人脸匹配
3. 已注册客人 → `欢迎回来，[姓名]先生/女士！`
4. 陌生来宾 → `您好，欢迎光临！请问有什么可以帮您？`
5. 30分钟内不重复问候同一个人

## 注意事项

- 如果 `openclaw.json` 中 `camera.enabled` 为 `false`，此技能不活跃
- 本地文字对话测试不受此技能影响
