# 🎉 折扣活动管理 (Promotion Management)

## 触发条件

当用户提到：活动、促销、折扣、打折、优惠、立减、连住、创建活动、查看活动。

## 工作流程

### 展示当前活动列表

先列出所有未删除的活动，然后给出操作选项：

```
🎉 当前促销活动

[活动列表]

请选择操作：
1️⃣ 创建新活动
2️⃣ 暂停/启用活动
3️⃣ 删除活动
4️⃣ 返回
```

### 操作 1: 创建新活动

1. **权限检查** — 需要 `promotion.create` 权限
2. 选择活动类型：
   - 1️⃣ 打折（如: 全场 8.5 折）
   - 2️⃣ 立减（如: 每晚立减 50 元）
   - 3️⃣ 连住免费（如: 住3免1）
   - 4️⃣ 免费升级（如: 标准间升豪华房）
3. 输入活动名称、描述、日期范围
4. 生成活动 ID（格式 `PROMO-XXX`）
5. 确认创建，通知前台 Agent 同步

### 操作 2: 暂停/启用活动

输入活动 ID，切换其状态（active ↔ paused）。

### 操作 3: 删除活动

输入活动 ID，**二次确认后**标记为已删除。

|------|------|------|------|------|
| 老板 | ✅ | ✅ | ✅ | ✅ |
| 总经理 | ✅ | ✅ | ✅ | ✅ |
| 运营经理 | ✅ | ✅ | ✅ | ❌ |
| 部门主管 | ✅ | ❌ | ❌ | ❌ |
| 普通员工 | ❌ | ❌ | ❌ | ❌ |

## 注意事项

- 删除操作需二次确认且不可恢复
- 创建/修改后提示"已通知前台 Agent 同步"
- 默认当前用户为 U001（张老板），拥有全部权限
